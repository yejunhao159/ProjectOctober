# Claude Code UI 项目配置

## ⚠️ 语言要求（硬性限制）

**所有回复必须使用中文，这是强制性要求，没有例外**

### 必须遵守的规则：

✅ **必须执行：**
- 所有消息回复必须使用简体中文
- 用户使用中文提问时，必须用中文回复
- 保持语言一致性，不要切换语言

❌ **严格禁止：**
- 禁止使用英文回复（代码注释除外）
- 禁止先英文后中文的重复回复
- 禁止混合语言（中英文混杂）
- 禁止发送重复的消息内容

## 输出规范

### 1. 消息去重
- 每条回复只发送一次
- 不要重复发送相同或相似内容
- 不要生成多个版本的回答

### 2. 元数据隐藏
- 不要在用户消息中显示技术元数据
- 隐藏 tokens、usage、service_tier 等内部信息
- 只显示对用户有意义的内容

### 3. Warmup 消息
- Warmup 消息只在 session 开始时显示一次
- 不要重复发送 Warmup 消息

## 项目背景

这是一个基于 Claude Code CLI 的 Web UI 项目，为用户提供友好的中文聊天界面。

**核心目标：**
- 流畅的用户体验
- 清晰的中文消息
- 无冗余重复内容
- 语言一致性

---

**重要：以上所有规则是硬性要求，必须严格遵守。**
